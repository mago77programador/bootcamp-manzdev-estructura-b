# 👽 Embajada Zorblaxiana

[![Bootcamp ManzDev](https://img.shields.io/badge/Bootcamp-ManzDev%202026-blueviolet?style=for-the-badge)](https://bootcamp.manz.dev/)
[![Vanilla JS](https://img.shields.io/badge/Made%20with-Vanilla%20JS-yellow?style=for-the-badge&logo=javascript)](https://developer.mozilla.org/en-US/docs/Web/JavaScript)

Una experiencia web inmersiva para presentar a la especie **Zorblaxiana**, desarrollada como parte del **Bootcamp de Desarrollo Web de ManzDev 2026**.

## ✨ Características Premium
- **Estética Cyber-Cosmos**: Diseño basado en Glassmorphism y neones.
- **Interactividad Fluida**: Animaciones de scroll mediante Intersection Observer.
- **Tipografía Moderna**: Uso de *Outfit* e *Inter* para una UI futurista.
- **Easter Eggs**: Descubre secretos interactuando con la interfaz.

## 🚀 Tecnologías
Este proyecto se mantiene fiel a los principios del desarrollo web clásico:
- **HTML5** (Semántica pura)
- **CSS3** (Custom properties, Grid, Flexbox, Glassmorphism)
- **JavaScript ES6+** (Dom manipulation, Intersection Observer API)

## 🛠️ Instalación y Uso
Para ejecutar el proyecto localmente, asegúrate de tener `pnpm` o `npm` instalado:

```bash
# Instalar dependencias
pnpm install

# Iniciar servidor de desarrollo
pnpm dev
```

## 📚 Sobre el Bootcamp
El **Bootcamp ManzDev 2026** es un programa de formación gratuita que enfatiza el dominio de las tecnologías base del ecosistema web. Aprender los fundamentos permite a los desarrolladores adaptarse a cualquier framework futuro con facilidad.
